Align

Documents of Different Sizes?

Need to compare a plan of entire floor and a plan of a separate room? You might need to use the Align feature to make you comparison precised.

Select Points

Select 2 points on the corners of area you want to compare on the first document and second document. The same points should be selected appropriately.

Check result

Once the last point on the second document is placed, pages will be automatically overplayed and the possible changes od selected areas highlighted with color.
